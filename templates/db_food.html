How to Use This Page

This page allows you to edit the food database.

Search & Edit Food: You can search for food by entering the food code or name.

Search Button: This allows you to retrieve the entire list.

Edit Food Button: This allows you to modify the selected food's code and name.

Add Food Button: This allows you to add a new food item to the database.

Delete Food Button: This allows you to remove the selected food from the database.

Edit Ingredient: You can add, modify, or delete ingredients for a specific food.

Edit Ingredient Button: This allows you to change the amount of a specific ingredient for 1 person (g).

Add Ingredient Button: This allows you to add an ingredient to a specific food.

Delete Ingredient Button: This allows you to remove the selected ingredient from the specific food.
