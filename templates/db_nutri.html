How to Use the Page

This page allows you to edit the food ingredient database.

Search & Edit Ingredient: You can search for food ingredients by entering their codes or names.

Search Button: This allows you to retrieve the entire list.

Edit Ingredient Button: This allows you to modify the selected food ingredient's code and name.

Add Ingredient Button: This allows you to add new food ingredient item to the database.

Delete Ingredient Button: This allows you to remove the selected food ingredient from the database.

Edit Nutrient: You can edit and check the nutrients of the food ingredients.

Save Nutrient Button: This allows you to save the edited nutrients of the food ingredients.
